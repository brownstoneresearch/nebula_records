## Google Search Console setup

1. Open Google Search Console.
2. Add the property: `https://nebularecordholdings.art/`.
3. Get your HTML meta verification code.
4. Replace `PASTE_GOOGLE_SEARCH_CONSOLE_VERIFICATION_CODE_HERE` inside public HTML pages.
5. Submit this sitemap: `https://nebularecordholdings.art/sitemap.xml`.
6. Use URL Inspection to request indexing for the homepage and main public pages.
