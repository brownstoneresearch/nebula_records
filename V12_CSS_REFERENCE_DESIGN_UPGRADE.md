# V12 CSS Design-System Upgrade

This version analyses the uploaded CSS reference and applies its useful design ideas to the Nebula Records site without copying the full generated utility bundle.

Applied ideas:
- Plus Jakarta Sans for interface typography, Playfair Display for premium headings, and Geist Mono for technical labels.
- HSL-like theme tokens, elevated shadows, primary teal and amber accent palette.
- Dark card surfaces with strong text contrast for readability.
- Noise overlay, soft focus rings, reduced motion support, and mobile-first responsive overrides.
- Cleaner header logo contrast, premium footer, improved form fields, dashboard tables, cards and preview shelf styling.
